# 画像ボタンに割り当てる関数リスト

業務担当者が画像ボタンから実行する関数の推奨リストです。

---

## 🎯 最小構成（2つのボタンのみ）

### 1. ワンクリック初期設定ボタン（初回のみ）🎉

**関数名**: `completeInitialSetup`

**ボタン名の例**: 「🎉 初期設定」「初回設定スタート」「ワンクリック設定」

**用途**:
- **このボタン1つで初期設定がすべて完了します！**
- 実行内容:
  1. 権限承認（スプレッドシート、Googleドライブ、外部URL）
  2. onOpenトリガー設定（メニュー自動表示）
  3. 設定検証
- 初回利用時に1回だけ実行

**配置場所**: リサーチシートの最上部・最も目立つ場所（例: A1セル）

**実行頻度**: 初回のみ

**重要度**: ⭐⭐⭐⭐⭐ 最重要

**ユーザーへの説明**:
```
【初回のみ】このボタンを押すだけで初期設定が完了します
1. ボタンをクリック
2. 表示されるダイアログで「許可」をクリック
3. 完了メッセージが表示されたら、ページをリロード（F5キー）
4. 以降は「出品」ボタンのみ使用
```

---

### 2. 出品ボタン（メイン機能）

**関数名**: `transferListingData`

**ボタン名の例**: 「📦 出品」「eBayに出品」「転記実行」

**用途**:
- リサーチシートのデータを出品シートに転記
- eBay APIから商品情報を取得
- 画像をダウンロードしてGoogleドライブに保存
- Item Specificsを優先度順にソート・色分け

**配置場所**: リサーチシートの目立つ場所（例: A3セル）

**実行頻度**: 毎回の出品作業で使用

**重要度**: ⭐⭐⭐⭐⭐ 最重要

## 📌 オプションボタン（メニューから実行すればOK）

以下の機能は画像ボタンにする必要はありません。
メニュー「初期設定」から実行できます。

### 設定確認・検証

**関数名**: `showConfig` / `checkConfig`

**メニュー**: 「初期設定」→「設定を表示」「設定を検証」

**用途**:

**実行頻度**: 設定確認・変更時

## 🎨 画像ボタン作成の推奨

### ボタンデザインのポイント

1. **サイズ**: 200×50 ピクセル程度
2. **色分け**:
   - 🔴 赤色系 = 重要/必須（権限承認、出品）
   - 🔵 青色系 = 設定/確認
   - 🟡 黄色系 = オプション機能
3. **アイコン**: 絵文字またはアイコン画像を使用
4. **テキスト**: 大きく読みやすいフォント

### PowerPointでの作成例

```
1. PowerPointで新規スライド作成
2. 図形（長方形）を挿入
3. サイズ調整: 幅 5cm、高さ 1.5cm
4. 塗りつぶし: グラデーション（青→水色など）
5. テキストボックス: 「📦 出品」など
6. 画像として保存（PNG形式推奨）
```

## ⚙️ 画像ボタンの設定手順

### 手順1: 画像を挿入

1. リサーチシートを開く
2. メニュー「挿入」→「画像」→「セルに挿入」
3. 作成した画像を選択

### 手順2: スクリプトを割り当て

1. 挿入した画像をクリックして選択
2. 画像を**右クリック**
3. 「スクリプトを割り当て」を選択
4. 関数名を入力（例: `authorizePermissions`）
5. OKをクリック

### 手順3: 動作確認

1. 画像をクリック
2. 関数が実行されることを確認

## ⚠️ 重要な注意事項

### 1. 初期設定ボタンは初回に1回だけ実行

- 初回利用時に「🎉 初期設定」ボタンを1回押すだけ
- このボタン1つで、権限承認・トリガー設定・検証がすべて完了
- 2回目以降は実行不要です

### 2. 関数名のスペルミスに注意

- 関数名は**完全一致**である必要があります
- 大文字・小文字を正確に入力してください
- コピー＆ペーストを推奨

### 3. 初期設定後は「出品」ボタンのみ使用

- 初期設定が完了したら、「📦 出品」ボタンだけを使用
- 設定確認・検証はメニューから実行できます

## 🎯 最小構成（2つのボタンのみ！）

**最高にシンプル！**

| ボタン | 関数名 | 用途 | 使用タイミング |
|--------|--------|------|-------------|
| 🎉 初期設定 | `completeInitialSetup` | 権限承認+トリガー設定+検証を一括実行 | 初回のみ（1回だけ） |
| 📦 出品 | `transferListingData` | リサーチシートから出品シートへ転記 | 毎回使用 |

**初回設定後は、「📦 出品」ボタンのみを使用すればOKです。**

### なぜ2つで十分なのか？

1. **`completeInitialSetup` が全部やってくれる**
   - 従来の `authorizePermissions` + `setupOnOpenTrigger` を統合
   - 1回実行するだけで初期設定完了

2. **設定確認・検証はメニューから実行できる**
   - メニュー「初期設定」→「設定を表示」
   - メニュー「初期設定」→「設定を検証」

3. **メニューは自動表示される**
   - `completeInitialSetup` でonOpenトリガーが設定される
   - 次回からスプレッドシートを開くと自動でメニュー表示
